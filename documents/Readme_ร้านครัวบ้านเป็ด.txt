    Setup
1.นำไฟล์ Zip ที่ โหลดมาเเตกไฟล์ไปใน Drive c/xampp/htdocs
2.เปิด Xapp ขึ้นมา Start My SQL และ Apeche
3.เปิดไฟล์ WP-config ใน Editor ใดก็ได้ เ
แก้ไข DB_name (Database name) เป็น banped
แก้ไข DB_USER เป็น root
แก้ไข DB_PASSWORD เป็นช่องว่าง
4.เปิด Browser ขึ้นมา พิมพ์  Localhost or 127.0.0.1 /project/wp-admin 
5.ทำการติดตั้ง wordpress  เลือกภาษา ใส่ข้อมูลที่เเก้ไขไปเมื่อครู่
6.ทำการ import ฐานข้อมูล SQL ตั้งชื่อ DB_name เป็น banped
